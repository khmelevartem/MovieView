MovieView

Приложение для отображения списка фильмов.

Сплэш скрин **анимирован**.

Использует **пагинацию** - дозагрузка происходит при пролистывании до конца имеющегося списка.

- kotlin
- Retrofit
- RxJava
- RecyclerView
- mvvm-mvi
