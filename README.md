MovieView

Application that shows you a list of the most recent movies.

Splash screen is animated.

Uses paging (custom). New info is downloaded when you scroll to the pre-last item

Technologies:
- kotlin
- Retrofit
- RxJava
- RecyclerView
- mvvm-mvi
